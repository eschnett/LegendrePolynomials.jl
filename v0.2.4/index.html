LegendrePolynomials.Pl! — Method
Pl!(arr::AbstractArray, x::Real; [lmax = maximum(axes(arr,1))])

Computes the Legendre Polynomials $P_l(x)$ for the argument x and l = 0:lmax, and saves it in arr. Assumes that arr has 0-based indexing.

All dimensions of arr should have indices starting from 0. The axes of arr should be (0:l, 0:n) where l >= lmax and n >= 0

At output, arr[l,0] == Pl(x,l)

The optional keyword argument lmax may specify the range of $l$'s to compute. It defaults to lmax = maximum(axes(arr,1))

LegendrePolynomials.d2Pl! — Method
d2Pl!(arr::AbstractArray, x::Real; [lmax = maximum(axes(arr,1))])

Computes the second derivatives of Legendre Polynomials $d_x^2P_l(x)$ for the argument x and l = 0:lmax, and saves them in arr. Assumes that arr has 0-based indexing.

The first dimension of arr should be 0:l, where l >= lmax The second dimension – if arr is 2D – should contain the index 2

At output, arr[l] == d2Pl(x,l) if arr is a Vector or arr[l,2] == d2Pl(x,l) if arr is a Matrix

The optional keyword argument lmax may specify the range of $l$'s to compute. It defaults to lmax = maximum(axes(arr,1))
